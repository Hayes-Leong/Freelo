## v2.2.0 (2026-07-20)

### 🏆 成就系统
- 独立成就面板，三张渐变成就卡片：🔥连续打卡天数、🍅累计番茄、🍌累计香蕉
- 里程碑自动检测：番茄达人、勤劳香蕉等称号

### ⚙️ 设置面板
- 独立设置面板：番茄专注/休息时长、打卡奖励、时区选择
- 香蕉换算公式可配：默认每 25 分钟工作 = 1 根香蕉
- 时区支持：UTC-12 到 UTC+9，面板实时显示选定时间

### 🎨 图标系统
- 新增 SvgIcon.vue 统一图标组件（clock/briefcase/book/chart/award/settings/check/zap/calendar）
- 全界面 emoji 替换为笔画图标

### 📋 代办看板搬家
- 灵感代办看板从右栏移至左栏，与 OKR 树 Tab 并列

### 🍌 香蕉计算逻辑变更
- 旧：每段手动计时 = 1 根香蕉
- 新：手动计时总分钟 ÷ 可配阈值 = 香蕉数

## v2.1.0 (2026-07-13)

### 🍅 番茄钟模式
- 秒表面板新增 🍌香蕉钟 / 🍅番茄钟 模式切换
- 番茄模式：专注倒计时（默认 25min）→ 自动记录 → 休息倒计时（默认 5min）

- Stats API 新增 achievements 端点
- Calendar API 新增 week 端点

## v2.0.0 (2026-07-08)

### 🎨 UI 全面翻新
- 设计风格由默认 Tailwind 模板升级为温润日式风格
- 配色方案：米白底 #fdfaf5、暖灰文字 #4a3f35、稻黄 #d4a853、墨绿 #5b8c5a、陶土 #c17f59
- 标题使用 Playfair Display 斜体艺术字体
- SVG 图标替代 emoji，整体更加克制精致

### ⏱️ 柳比歇夫时间管理法
- 任务类型：每个待办分为「工作」或「学习」（独立于 OKR）
- 预估时间：创建待办时必填，15 分钟为最小单位
- 秒表正计时：点击待办旁的 ▶ 开始计时，支持暂停/继续/停止
- 时间分段：每次暂停→继续产生一个独立分段，完整保留
- 后台运行：关闭浏览器后计时继续，刷新页面自动恢复

### 📊 时间统计模块
- 今日总结：当日总耗时、工作/学习分别统计
- 趋势折线图：ECharts 渲染，支持周/月/季时间跨度切换

### 🏗️ 前端工程化
- 从单文件 487 行 HTML 重构为 Vue 3 + Vite 组件化架构
- 17 个 SFC 组件，3 个 composable
- Tailwind CSS 本地构建，自定义温润日式色板

### 🔧 后端扩展
- 新增 timer/stats 路由
- 扩展 Todo 数据模型
- 新增 time_records 表
- 旧数据自动迁移
