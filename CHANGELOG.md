# 更新日志

## v2.0.0 (2026-07-08)

### 🎨 UI 全面翻新
- 设计风格由默认 Tailwind 模板升级为**温润日式**风格
- 配色方案：米白底 `#fdfaf5`、暖灰文字 `#4a3f35`、稻黄 `#d4a853`、墨绿 `#5b8c5a`、陶土 `#c17f59`
- 标题使用 Playfair Display 斜体艺术字体
- SVG 图标替代 emoji，整体更加克制精致

### ⏱️ 柳比歇夫时间管理法
- **任务类型**：每个待办分为「工作」或「学习」（独立于 OKR）
- **预估时间**：创建待办时必填，15 分钟为最小单位
- **秒表正计时**：点击待办旁的 ▶ 开始计时，支持暂停/继续/停止
- **时间分段**：每次暂停→继续产生一个独立分段，完整保留
- **后台运行**：关闭浏览器后计时继续，刷新页面自动恢复
- **级联删除**：删除待办时自动删除关联的时间记录

### 📊 时间统计模块
- **今日总结**：当日总耗时、工作/学习分别统计
- **趋势折线图**：ECharts 渲染，支持周/月/季时间跨度切换
- **渐变面积填充**：温润配色，圆角 tooltip，时间格式化为 `1h37m`
- **智能空状态**：无数据时显示最近记录日期和引导文案

### 🏗️ 前端工程化
- 从单文件 487 行 HTML 重构为 **Vue 3 + Vite** 组件化架构
- 17 个 SFC 组件，3 个 composable（useApi / useTimer / useToast）
- Tailwind CSS 本地构建，自定义温润日式色板
- ECharts 5 图表集成

### 🔧 后端扩展
- 新增 `routes/timer.js` — 秒表状态机 API（start/pause/resume/stop/current）
- 新增 `routes/stats.js` — 统计查询 API（daily/range）
- 扩展 Todo 数据模型：`task_type`、`estimated_duration`、`total_elapsed`
- 新增 `time_records` 表和 `_timer_state` 全局状态
- 旧数据自动迁移，无需手动处理

### 📁 项目结构变更
```
v1.0                          →  v2.0
public/index.html (487行)     →  src/ (17个.vue组件)
Tailwind CDN                   →  Tailwind 本地构建
无构建工具                     →  Vite 5
```
