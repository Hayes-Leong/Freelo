# Freelo RPG 养成系统 — 游戏策划案

> 版本：v1.0 草案 | 状态：策划阶段，暂不开发

---

## 一、核心循环

```
┌─────────────────────────────────────────────────────┐
│                                                     │
│   计时工作 ──→ 获得金币 ──→ 买武器 ──→ 打怪更轻松    │
│      │                      │            │          │
│      │ 完成番茄/香蕉        │            │ 打赢      │
│      └──→ 消耗换经验 ───────┘            │          │
│                │                         ↓          │
│                ↓                      怪物升级      │
│             升级加属性点 ←────────────────┘          │
│                                                     │
└─────────────────────────────────────────────────────┘
```

一句话：**你越高效工作，角色越强；角色越强，打怪越爽；怪物越来越强，逼你持续工作变强。** 正反馈闭环。

## 二、角色属性

### 基础面板

| 属性 | 说明 | 初始值 | 来源 |
|------|------|--------|------|
| Lv | 等级 | 1 | 经验值满自动升级 |
| EXP | 经验值 | 0/100 | 消耗 🍅 或 🍌 兑换 |
| ⚔️ 力量 | 影响攻击力 | 3 | 每升 1 级 +1 自由分配点 |
| 📚 智力 | 影响暴击率 | 3 | 每升 1 级 +1 自由分配点 |
| ❤️ 生命 | 战斗血量 | 20 | 随等级自动成长 (+5/Lv) |
| 💰 金币 | 货币 | 0 | 计时分钟数 1:1 转换 |
| 🔥 连击 | 连续打卡天数 | 0 | 自动累计，断签归零 |

### 属性点分配规则

- 每升 1 级获得 **1 点自由属性**，可在角色面板手动加到力量或智力
- 力量：每点 +2 攻击力
- 智力：每点 +3% 暴击率（暴击 = 伤害×2）

### 经验值公式

| 消耗 | 获得经验 |
|------|---------|
| 1 个 🍅 番茄 | 50 EXP |
| 1 根 🍌 香蕉 | 20 EXP |

- 升级经验曲线：`Lv × 100`（Lv1→2 需 100，Lv2→3 需 200，以此类推）
- 经验满后需**手动确认升级**（弹窗选加点方向）

## 三、货币系统

| 货币 | 获取方式 | 用途 |
|------|---------|------|
| 💰 金币 | 计时分钟 1:1 兑换（例如工作 90min = 90 金币） | 买武器、买药水 |
| 🍅 番茄 | 完成一个完整的番茄专注（25min） | 换经验 (50 EXP) |
| 🍌 香蕉 | 手动计时每满阈值分钟（默认 25min） | 换经验 (20 EXP) |

**设计意图**：金币让你买装备（短期快感），番茄/香蕉让你想升级（中期目标），升级加点让你感觉成长（长期绑定）。

### 商店交互

- 放在角色面板内，一个"🏪 武器商店"按钮
- 点击打开武器列表，已拥有的标注"✅ 已装备"，买得起的标注价格，买不起的灰掉
- **一次只能装备一把武器**
- 卖出旧武器：半价回收
- 新武器买到后自动装备

## 五、战斗系统 ⚔️

### 怪物生成

| 参数 | 规则 |
|------|------|
| 刷新频率 | 每完成 3 个番茄或累计 2 小时计时，刷新一只怪物 |
| 等级 | 怪物等级 = 玩家等级 ± 2（随机浮动） |
| 属性 | `HP = 10 + 怪物等级 × 8`，`ATK = 2 + 怪物等级 × 2` |
| 名字池 | Lv1-5：史莱姆/野鼠/树精/石像鬼，Lv6-15：骷髅兵/暗影狼/熔岩龟，Lv16+：死灵法师/冰霜巨龙/暗影领主 |

### 战斗公式

```
玩家攻击力 = 力量 × 2 + 武器攻击力
暴击判定 = 随机(0-100) < 智力 × 3%，暴击则伤害×2
连击倍率 = 1 + 连续打卡天数 × 0.05（7天连击 = 1.35倍伤害）

玩家每回合伤害 = 玩家攻击力 × 连击倍率（暴击则×2）
怪物每回合伤害 = 怪物ATK

战斗回合数 = ceil(怪物HP / 玩家每回合伤害)
玩家受伤 = 怪物ATK × 战斗回合数
```

### 战斗结果

| 结果 | 条件 | 奖励 |
|------|------|------|
| 🏆 胜利 | 玩家伤害 ≥ 怪物HP | +金币(怪物等级×20)，+EXP(怪物等级×10)，可能掉落稀有物品 |
| 💀 战败 | 玩家伤害 < 怪物HP 且 玩家HP不够 | 不掉血，但拿不到奖励，建议升级/换武器再来 |

### 自动战斗

- 战斗**不需要玩家操作**（挂机流）
- 点击"⚔️ 讨伐"按钮 → 3 秒动画（武器图标碰撞怪物图标 + 伤害数字飘出）→ 显示结果
- 赢了：怪物素材收进背包，弹出奖励
- 输了：怪物嘲讽一下，提示"力量不足，建议升级或更换武器"

## 六、连击系统 🔥

| 连续打卡 | 倍率 | 效果 |
|---------|------|------|
| 1-2 天 | ×1.00 | 正常 |
| 3-6 天 | ×1.15 | 连击开始 |
| 7-13 天 | ×1.35 | 🔥 周连击 |
| 14-29 天 | ×1.50 | 🔥🔥 双周连击 |
| 30+ 天 | ×2.00 | 🔥🔥🔥 月度传说 |

- **断签惩罚**：连击归零，倍率重置
- **连击保护**：休息日不计入断签（你设的休息日不算偷懒）

## 十一、设计原则

1. **挂机不肝**：战斗不操作，3 秒出结果。你不是来做游戏代练的。
2. **不惩罚**：输了不扣金币不掉血，只是拿不到奖励。鼓励而非惩罚。
3. **与工作效率正相关**：你越高效 → 角色越强。不存在"氪金"通道，唯一货币是时间。
4. **渐进解锁**：前 3 天就能体验到完整循环（打史莱姆→升级→买棍子），避免"玩了三天什么都没解锁"的挫败感。
